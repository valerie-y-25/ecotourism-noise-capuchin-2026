# Ecotourism Noise Disrupts Foraging in Two Capuchin Monkey Species (*Cebus albifrons* and *Sapajus apella*)

This repository contains code, data, and figures used in the manuscript.

## Contents
- `data` - cleaned focal and scan datasets from audio recordings
- `exploratory_analysis` - older scripts and exploratory tests
- `figures` - output figures used in manuscript
- `primary_analysis` - main focal and scan analysis scripts referenced in manuscript
- `species_specific_analysis` - species comparison, individual models run using a single species

## Main scripts in `primary_analysis`
- `Focal Data Analysis`
- `Scan Data Analysis (new, revised poisson).R`
- `species_effect_analysis.R`
- `ICC Test.R`

## Notes
- Several older scripts are retained in this repository to verify intermediate steps and include earlier model checks.
- This repository is still in active revision alongside the manuscript. 
- Final results reported in manuscript are based on the scripts listed above under `primary_anlysis`.
